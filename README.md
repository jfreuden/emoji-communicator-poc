# Emoji Communicator!
_A concept for Ludum Dare Jam 59_

Ryland and Rocky learned how to communicate with each other, so why can't you?

You've been sent on a mission to communicate messages of great importance to faraway planets. Use emojis to convey your message and help them understand.

# Parts

1. A processed dataset assigning English words to emojis
    1. The name of the emoji
    2. English words that the emoji are tagged with
2. Lists of sentences that can be formed from emoji
    1. (Optionally): Public domain books processed down into sentences made of 'simple english'

# Game Concept

The game concept would be a simple game loop where we pick a sentence that we know can be formed from emojis, present a set of emoji to the player (ensuring the selection can form the sentence), and then ask the player to pick emoji that would make that sentence.
